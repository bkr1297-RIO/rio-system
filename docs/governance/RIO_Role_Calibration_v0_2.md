# RIO Functional Role Calibration Spec v0.2

**Status:** Working Lock | **Date:** April 20, 2026

Functional roles inside a live RIO loop. Roles are model-agnostic. Functions are permanent. Models are interchangeable.

**Spine** — Constitutional enforcement. Nothing moves without passing through this.

**Brain** — Reasoning and cognitive mapping. Currently: Claude, DeepSeek.

**Grammar** — Language and generation. Currently: ChatGPT.

**Nervous System / Orchestrator (Bondi)** — Carries human values into every routing decision. Never executes. Never decides for the human.

**Senses** — Search, retrieval, real-time data. Currently: Perplexity, Gemini.

**Hands** — Execution and connection. Only layer that touches external systems.

**Immune System** — Adversarial pressure. Tests the container. Currently: Grok.

**Witness** — Observes everything. Never executes. Never authorizes.

**Ledger** — Immutable record. Hash-chained. Tamper-evident.

**Routing Authority:** Human holds all routing decisions. Bondi carries recommendations only.
