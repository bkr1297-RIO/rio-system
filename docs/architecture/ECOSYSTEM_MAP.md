## Where RIO Sits in the Industry

RIO does not compete with Google, OpenAI, Microsoft, or any other AI provider. RIO sits between them as the governance and control layer.

```
Users
  |
  v
ONE (Interface / Operating System)
  |
  v
RIO (Governance / Policy / Approval)
  |
  v
Gateway (Execution Router)
  |
  +-- Google (Gmail, Drive)
  +-- Microsoft (Outlook, Office)
  +-- Slack
  +-- Salesforce
  +-- Stripe
  +-- Any API
  |
  v
Ledger + Receipts (Proof)
```

## The Multi-Router Model

The system contains multiple routing layers, each responsible for a different type of routing.

| Router Type | Component | What It Routes |
|---|---|---|
| Intent Router | ONE | User intent |
| Task Router | Chief of Staff | Tasks and assignments |
| Policy Router | RIO | Governance decisions |
| Execution Router | Gateway | API calls and actions |
| Network Router | Internet | Packets |
