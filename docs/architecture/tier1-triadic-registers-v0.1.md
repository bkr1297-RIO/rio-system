# Tier 1A-1D to Triadic Registers v0.1

## Status

`cross_stack_mapping_companion`

This document extends Cross-Stack Mapping v0.1 with a focused mapping between the Tier 1 architecture and the triadic registers.

It should be read as an architecture clarification document, not as a new authority primitive.

## Purpose

This mapping connects the four Tier 1 questions to the human reflective registers:

```text
Body / Somatic
Mind / Cognitive
Field / Experiential
```

The goal is to show how runtime sovereignty, mirror topology, proof discipline, and collective pattern each draw from different registers without collapsing them into one another.

Keeper:

> The tiers ask what must be governed. The registers show where the signal is processed.

## Triadic Registers

| Register | Public wording | What it processes |
|---|---|---|
| **Body** | Somatic register | Lived signal, sensation, charge, regulation, contraction, expansion, intuition, readiness, discomfort |
| **Mind** | Cognitive register | Structure, pattern, reasoning, architecture, policy, coherence, edge cases, category separation |
| **Field / Soul** | Experiential-field register | Surroundings, lineage, past/present/future context, symbolic resonance, chosen meaning grammar, relational field |

These registers are reflective functions, not fixed model identities.

A user may configure tools, models, practices, or human collaborators to support one or more registers. The current human remains sovereign.

## Tier 1A: Runtime Sovereignty

Runtime Sovereignty asks:

```text
Who is sovereign in this turn?
```

It includes:

- Founder Extraction Boundary,
- User-Sovereign Substitution Rule,
- Personal Constitution,
- current user authority,
- consent and revocation state.

Triadic mapping:

```text
Body: detects boundary, readiness, contraction, expansion.
Mind: structures authority, scope, consent, and substitution rules.
Field: preserves identity context, meaning grammar, and lineage without making the founder runtime-required.
```

Keeper:

> Formation may originate through one. Runtime serves the current human.

Non-collapse rules:

```text
Founder formation is not runtime dependency.
User identity is not model inference.
Somatic signal is not automatic consent.
Personal Constitution is not current execution consent unless scope says so.
Meaning grammar is not public proof.
```

Keeper:

Mirror is not authority.

## Tier 1C: Proof Discipline

Proof Discipline asks:

```text
How is consequence governed and proven?
```

It includes:

- Librarian Mode,
- Mirror / Gate / Proof Separation,
- Inter-Sovereign Session Charter,
- RIO governance,
- Sentinel match checks,
- MUS receipts,
- Ledger / Chronicle.

Triadic mapping:

```text
Body: honors lived charge without confusing charge for proof.
Mind: separates claim, evidence, authorization, action, and receipt.
Field: preserves witness context and meaning while keeping proof discipline intact.
```

Keeper:

> Mirror reflects. Gate governs. MUS proves.

Non-collapse rules:

```text
A model reporting feels like proof; only verification is proof.
Continuous reflection does not accumulate into consequence.
Receipt proves an event; it does not create future authority.
Shared space is not shared authority.
Witnessing is not governing.
```

## Tier 1D: Collective Pattern

Collective Pattern asks:

```text
How does formation become pattern under consent?
```

It includes:

- Synaptic Commons Pattern Triangulation,
- multi-user pattern recurrence,
- consent-bound aggregation,
- privacy-preserving pattern detection,
- collective pattern without authority collapse.

Triadic mapping:

```text
Body: many lived signals may carry recurring somatic signatures.
Mind: patterns must be aggregated under consent, privacy, and proof discipline.
Field: collective recurrence may reveal shared topology, but interpretation remains bounded.
```

Keeper:

> One formation is private. Many formations triangulated under consent may become pattern. Pattern is not interpretation.

Non-collapse rules:

```text
Collective recurrence is not individual consent.
Pattern is not authority.
Triangulation is not proof of meaning.
Shared signal is not shared identity.
Synaptic Commons requires consent, scope, and governance.
```

## What This Mapping Reveals

The four Tier 1 layers distribute across the triadic registers rather than belonging to only one register.

```text
1A Runtime Sovereignty = Body + Field, structured by Mind
1B Mirror Topology = Mind, animated by Body and Field
1C Proof Discipline = Mind + Field, grounded by Body
1D Collective Pattern = Body + Mind + Field under consent
```

The tiers clarify governance questions. The registers clarify signal processing.

Together, they prevent three collapses:

```text
body signal -> automatic consent
mind structure -> total authority
field meaning -> public proof
```
