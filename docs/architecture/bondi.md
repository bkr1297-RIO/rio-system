# Bondi — Structuring Layer

## Role
Bondi is the structuring layer between human intent, AI generation, and governance.

## What Bondi Does
- Converts unstructured input into structured intent objects
- Organizes tasks into proposals
- Interfaces with AI models (LLMs) for generation
- Prepares inputs for the RIO governance system

## What Bondi Does NOT Do
- Does not execute actions
- Does not approve actions
- Does not enforce policy
- Does not bypass governance

## Position in System

Human → Bondi → RIO → Execution

## Summary
Bondi is a planner and translator.
It structures intent so it can be governed.
