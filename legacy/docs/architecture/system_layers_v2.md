# System Layers (vNext)

## Overview
This update formalizes the separation of system responsibilities across six layers.

## Layers

- **Atlas (SAS)** — Awareness Layer  
  Observes patterns and state. Zero authority.

- **Bondi** — Structuring Layer  
  Translates input into structured intent.

- **RIO** — Governance Layer  
  Evaluates actions using policy, risk, and approval.

- **MANTIS** — Memory & Learning Layer  
  Stores patterns, state, and outcomes.

- **Ledger** — Proof Layer  
  Cryptographic record of all governed actions.

- **ONE** — Interface Layer  
  User interaction surface.

## Principle

No layer may assume the role of another.

- Observation ≠ Decision  
- Decision ≠ Execution  
- Execution ≠ Proof
